Welcome to my Web Page!

I am second year Masters student in Department of Computer Science at University of California, Santa Barbara. My research interests include Deep Learning, Computer Vision, Data Engineering, Psychology and Gender Studies. I have worked in a variety of projects involving applications of these areas to real world problems. I am currently working on my Masters thesis under the supervision of Professor Arpit Gupta, Systems and Networking Lab, UCSB. As part of this effort, I built a robust and efficient network data processing framework that is capable of dealing any kind of networking data and processing it any format based on user requirement. Used this framework to synthesize multiple datasets here at SNL lab, UCSB. Apart from this, I am also involved in the network data representation project that aims at creating a representation learning framework capable of learning network data during the pre-training stage using a hierarchical attention model and can be finetuned on any down stream task. Please look into the projects section to see all the projects I have worked on.

Research

  • Network Data Processing Pipeline Framework(Jul ’22) Developed an efficient data processing pipeline framework for networking data that is robust, generalizable and extendable to any use case based on requirement. Capable of cleaning, processing and converting raw packet capture data into polished feature sets of multiple granularity (packet, burst and flow) swiftly while being fault tolerant even on very large datasets. Our efficient framework also takes up the ardent task of labelling network data with ease.
  • Network Data Representation(Aug ’22) Contributed to the data processing and labeling part of the Network Data Representation project. (Details TBD)
  • Contextualization of Speed test measurements using Attention based models.(Dec ’22) Undertook a project of building data processing pipelines for the project contextualizing speed test measurements using attention models. Wrote pipeline scripts that can query data from google cloud using big query and process measurement data seperating huge segement into a single measurement granualrity (upload + download test).
  • The emission and transmission probabilities play a crucial role in deciding the state of the channel and prediction of these statistics to understand the state of channel only by looking at the packet drop statistics has been a long debated issues. Developed a neural network model to predict the emission and transmission probabilities in the Gilbert Elliot channel. Collected packet drop statistics using Sim2net package python as input for the model.Reported ̃75% accuracy which was better than existing state of the art Baum Welch that gave ̃55% accuracy.Paper accepted at Indian Academy of sciences Summer Research Fellow reports journal
